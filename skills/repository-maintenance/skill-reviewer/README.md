# Skill Reviewer

## Purpose

Provide a repeatable human review process that prevents vague, duplicative, unsafe, and unvalidated Skills from entering the library.

## Skill Files

- `SKILL.md`: Review workflow and decision rules.
- `CHECKLIST.md`: Review gate.
- `BEST_PRACTICES.md`: Review techniques.
- `COMMON_MISTAKES.md`: Review failures.
- `TROUBLESHOOTING.md`: Recovery guidance.
- `manifest.json`: Registry metadata.

## Example

Review the diff and complete `templates/reviews/skill-review.md`; run the generator checks and validator before recording approve, request changes, or reject.

## Validation

Every approval must include findings, validation evidence, and explicit residual-risk status.
