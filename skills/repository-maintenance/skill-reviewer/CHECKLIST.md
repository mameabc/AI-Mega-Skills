# Skill Reviewer Checklist

## Scope and Duplication

- [ ] The Skill solves one clear problem.
- [ ] Existing Skills and the registry were checked for overlap.
- [ ] Triggers and non-goals prevent scope drift.

## Quality

- [ ] Inputs, outputs, workflow, decisions, and failure handling agree.
- [ ] Examples are realistic, reproducible, and evidence-based.
- [ ] Best practices, common mistakes, and troubleshooting are useful.
- [ ] References are official or their limitations are explained.

## Repository Compliance

- [ ] All required files exist.
- [ ] Manifest values match the path and schema expectations.
- [ ] Generated registry and README outputs are current.
- [ ] Repository validation passes.

## Decision

- [ ] Findings are ordered by severity and include paths.
- [ ] Blocking findings are resolved before approval.
- [ ] The review records residual risks and a clear decision.
