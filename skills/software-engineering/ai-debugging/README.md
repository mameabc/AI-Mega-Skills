# AI Debugging

This Skill teaches AI coding agents a disciplined workflow for reproducing, localizing, explaining, fixing, and preventing real-world software failures.

## Included

- `SKILL.md`: workflow, decision trees, specialized debugging strategies, and a production template.
- `CHECKLIST.md`: repeatable investigation and verification checklist.
- `BEST_PRACTICES.md`: evidence, experiment, and remediation guidance.
- `COMMON_MISTAKES.md`: failure modes that make debugging slower or less reliable.
- `TROUBLESHOOTING.md`: guidance when evidence, reproduction, or tooling is incomplete.
- `examples/`: practical production debugging examples.

This Skill investigates runtime behavior. Use `ai-code-review` for evaluating a change without a reported failure, and use the prompt, context, or system-prompt Skills for agent instruction design.

Validate the repository with `pwsh scripts/validate-repository.ps1`.
